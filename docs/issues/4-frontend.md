# Issue #4: フロントエンド実装

### 背景 / 目的

ユーザーに認証フローの開始点（口座作成ボタン）と結果画面を提供する。最小限のUIで認証フローを体験できるようにする。

- 依存: #3
- ラベル: frontend

### スコープ / 作業項目

- `frontend/` ディレクトリ構成の作成
- トップ画面（index.html）: 口座作成ボタン
- 認証成功画面（callback.html）: ユーザー情報表示
- エラー画面（error.html）: エラーメッセージ表示
- TypeScript ロジック（app.ts）
- esbuild によるビルド設定

### ゴール / 完了条件（Acceptance Criteria）

- [ ] `frontend/` ディレクトリ構成が設計書通りに作成されている
- [ ] `index.html` に「口座作成」ボタンが表示される
- [ ] `callback.html` でクエリパラメータからユーザー情報を表示できる
- [ ] `error.html` でエラーコードに応じたメッセージを表示できる
- [ ] `npm run build` で `dist/` にビルド成果物が生成される
- [ ] CDKデプロイでS3にアップロードされ、CloudFront経由で表示できる

### テスト観点

- 検証方法: ブラウザで各画面の表示とクエリパラメータの処理を確認
